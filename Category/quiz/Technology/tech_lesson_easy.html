Technology Lesson (Easy)

  • WWW: "WWW" stands for World Wide Web, a system of interlinked hypertext documents accessed via the internet.
  • Computer Output: A monitor displays the output (visuals/text) from the computer.
  • Email Symbol: The "@" symbol is commonly used in email addresses.
  • Keyboard Use: A keyboard allows you to input text by typing.
  • Smartphone Type: A smartphone functions primarily as an input device for communication, apps, and browsing.
  • Printer Function: A printer is used to produce physical copies of digital documents.
  • Computer Brain: The CPU (Central Processing Unit) is considered the brain of the computer.
  • Cursor Control: A mouse moves the cursor on the screen to navigate or click items.
  • Picture File Format: ".jpg" is a common image file format for photos and graphics.
  • Wi-Fi Use: Wi-Fi allows devices to connect to the internet wirelessly.
