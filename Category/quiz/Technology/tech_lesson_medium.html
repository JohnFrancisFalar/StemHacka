• CPU (Central Processing Unit): This is the brain of the computer, responsible for processing all instructions.
  • F5 Key: Pressing F5 on most browsers refreshes or reloads the current web page.
  • Solid-State Storage: USB flash drives use solid-state memory, meaning no moving parts and faster access.
  • Microsoft: The Windows operating system was developed by Microsoft, one of the largest tech companies in the world.
  • Antivirus Software: This protects your computer by detecting and removing viruses, malware, and other threats.
  • HTML: Stands for Hyper Text Markup Language, used to structure web pages and their content.
  • Hard Drive: Stores data long-term, even when the computer is turned off. It can be HDD (magnetic) or SSD (solid-state).
  • Computer Virus: A harmful software program designed to spread and disrupt system operations or steal data.
  • Top-Level Domain (TLD): It’s the last part of a URL (.com, .org, .edu) that indicates the type of organization.
  • Copy Shortcut: Ctrl + C is a universal keyboard shortcut for copying selected text or items to the clipboard.
